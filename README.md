# Excel Data Analytics Portfolio

This repository contains Excel-based data analytics projects focused on
data cleaning, customer behavior analysis, and sales performance reporting.

## Projects

### 1. Retail Data Cleaning & Validation
- Cleaned raw retail sales data using Power Query
- Applied proper data typing and validation checks
- Prepared a reliable dataset for analysis

### 2. Customer Behavior Analysis Dashboard
- Aggregated transactional data to customer-level
- Segmented customers by value and risk
- Built an interactive Excel dashboard with KPIs and slicers

### 3. Sales Performance Dashboard
- Analyzed revenue trends, categories, and products
- Identified key revenue drivers and top-performing products
- Built an executive-style interactive dashboard

## Tools Used
- Microsoft Excel
- Power Query
- PivotTables & Pivot Charts
